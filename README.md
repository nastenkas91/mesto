# Project: Place
Interactive page where you can add/delete photos, open full-size view,
update profile information, add likes. All the forms are checked for validity.

## Technologies used:
* Adaptive layout
* BEM methodology
* OOP
* Responsive elements with JS
* Form validation
* Api
