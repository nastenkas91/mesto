# Проект: Место
Интерактивная страница, куда можно добавлять фотографии, удалять их, редактировать профиль и ставить лайки

## Используемые технологии:
  * адаптивная верстка
  * методология БЭМ
  * скрипты
  * ООП
  * валидация при помощи JS
